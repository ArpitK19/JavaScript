Introduction (Document Object Model)

	DOM is a way to represent the webpage in a structured hierarchical way so that it will become easier for programmer and users to guide through the document. with DOM we can easily access and manipulate tags, IDs, classes, Attribute or element of HTML using commands  or method provided by the document object.

Usind DOM, The JavaScript gets access to HTML as well as CSS of the web page and can also add behavior to the HTML element. so basically Document Object Model is an API that represent and interacts with HTML or XML Documents
